Usage
-----

Windows
~~~~~~~

1. Run **Command line(cmd)** as **Administrator**.
2. Execute the Shark Hunter:

.. code-block:: bash

  python h-shark.py -r pcapfile --scan
   
Linux
~~~~~

1. Give the permission to **the Shark Hunter script(h-shark.py)** Using **Terminal**:

.. code-block:: bash

  chmod+x h-shark.py 

2. Execute the Shark Hunter:

.. code-block:: bash

  ./h-shark.py -r pcapfile --scan
